CustomsBot

Run League of Legends custom-game series from Discord — series setup, live champion select, tournament codes, and post-game stats.

Add to Discord View source

What it does

Series setup

Create best-of series with two persistent teams, fearless draft, map, and region — all from a slash command.

Live champion select

A real-time web lobby and tournament-style draft: pick/ban order, swaps, ready checks, and side assignment.

Tournament codes

Generates a Riot tournament code per game and posts it to your channel with each side's locked-in champions.

Stats

Captures Match-V5 results automatically and serves series, player, and champion leaderboards.

How to use it

  1. Add the bot to your server Use the Add to Discord button above. Each server's series stay private to that server.
  2. Link your Riot account Players run /link-account so tournament codes and stats can find them.
  3. Create a series Run /create-series to set the name, best-of, draft type, map, and region.
  4. Open a draft & play Use /series new-draft to open the web lobby, ready up, run champion select, and start the game with the posted code.
  5. Review the stats Sign in to the web app with Discord to browse your servers' series, games, and leaderboards.
